# Blog Posts

Add `.md` files here, then run `npm run blog:import` to import into Supabase.

## Format

```
# Post #4 — Blog / Educational
**Publish Date:** March 17, 2026
**Pillar:** AI + Judgment
**Excerpt:** Short description for listing pages...
---

# How AI Is Changing Technical Accounting Research

Your article content in Markdown...
```

- **Publish Date** or **Suggested Publish Date:** Used for display when published
- **Pillar** or **Content Pillar:** Maps to category
- **Excerpt:** Short description for blog listing
- Title is taken from the first `#` heading after the `---` separator

## Workflow

1. Add `.md` files
2. Run `npm run blog:import`
3. Go to Admin → Blog Posts and toggle **Published** when ready
