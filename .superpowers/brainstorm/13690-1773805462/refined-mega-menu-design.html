  • Large icon boxes with drop shadow
  • Hardcoded #339CFF throughout
  • Company menu uses different layout
  • 750px Solutions, 900px Products — inconsistent
  • Strong blue highlight on active items
AFTER
  • Subtle near-white right panel (#fafbfc)

  • Smaller, tighter icon boxes
  • All colors via CSS variables

  • Consistent width across all menus (~700px)
  • Soft blue tint on active items (#f5f8ff)
